# Get-CohesityProtectionJobStatus

## SYNOPSIS
Gets the protection job status.

## SYNTAX

```
Get-CohesityProtectionJobStatus
```

## DESCRIPTION
Displays the protection job status for all jobs in the cluster.

## EXAMPLES

### EXAMPLE 1
```
Get-CohesityProtectionJobStatus
```
