# Set-CohesityView

## SYNOPSIS
Updates a Cohesity View.

## SYNTAX

```
Set-CohesityView -View <View> [<CommonParameters>]
```

## DESCRIPTION
Returns the updated Cohesity View.

## EXAMPLES

### EXAMPLE 1
```
Set-CohesityView -View $view
```
