# Pre-requisites

The pre-requisites for using the Cohesity PowerShell Module are as below:

* On Windows
  * Windows PowerShell version `5.1` or higher.
  * PowerShell Core `6.0` or higher.
* On Linux and Mac OS
  * PowerShell Core `6.0` or higher.
* Cohesity Cluster running version `6.0` or higher.
